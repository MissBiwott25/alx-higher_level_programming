# Classes and Objects

Project done during **Full Stack Software Engineering studies** at **alx_Africa**. It aims to learn about object oriented programming (OOP), classes, objects, instances, methods and attributes in **Python**.

## Technologies
* Scripts written in `Bash 5.1.16(1)`
* C files are compiled using `gcc (Ubuntu 11.2.0-19ubuntu1) 11.2.0`
* C files are written according to the `C90 standard`
* Python script are written in `Python 3.11. 1`
* Tested on `Ubuntu 22.04 LTS`
